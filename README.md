# Theme Blank
[![Software License][ico-license]](LICENSE.txt)

A blank concrete5 8+ theme

- html, head, body tags
- One div as page wrapper
- One area

## Installation
`$ composer require a3020/theme_blank`

`$ ./vendor/bin/concrete5 c5:package-install theme_blank`
